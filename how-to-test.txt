to start frontend server :
    cd 3dify --> npm run dev  

start flask backend server:
    cd 3Dify -->  cd backend --> python3 main.py 

HOW TO PASS STATE TO CHILD COMPONENT:
    - CREATE STATE IN PARENT COMPONENT 
    - ADD CHILD HTML TAG INTO PARENT RETURN STATEMENT WITH PROPS (ARGUMENTS)
    - RECEIVE THE STATE AS A PROP VIA PARAMETER IN CHILD COMPONENT 
    - NOW YOU CAN CALL THAT STATE IN THE FUNCTION AND USE IT
